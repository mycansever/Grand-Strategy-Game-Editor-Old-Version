Grand Strategy Game Editor – Old Version
English

This repository contains an old version of the Grand Strategy Game Editor, a development tool created to manage and edit core data of a grand strategy game project.

The editor was designed to simplify working with large datasets typical in strategy games, such as province data, map information, and gameplay configuration files. Instead of manually editing raw files, this tool provides a more organized way to modify game content.

Features

Province data editing

Strategy map data management

Editing of gameplay configuration files

Tools for working with large game datasets

Early UI systems for strategy game development

Purpose of this Repository

This repository preserves an early development version of the editor.
It may contain unfinished features, experimental systems, or legacy code that was later replaced or improved.

It is kept mainly for:

Archival purposes

Reference for development history

Studying the early architecture of the editor

Status

⚠️ Legacy / Deprecated Version

This version is no longer actively developed and is maintained mainly for historical reference.

Grand Strategy Game Editor – Eski Sürüm
Türkçe

Bu repository, Grand Strategy Game Editor aracının eski bir sürümünü içermektedir. Bu araç, büyük strateji oyunu projelerinde kullanılan temel veri dosyalarını düzenlemek ve yönetmek için geliştirilmiştir.

Editör; province verileri, harita bilgileri ve oyun ayar dosyaları gibi büyük veri yapılarını daha kolay düzenlemek amacıyla oluşturulmuştur. Böylece geliştiriciler ham dosyaları manuel olarak düzenlemek yerine daha düzenli bir araç kullanabilir.

Özellikler

Province verisi düzenleme

Strateji haritası verisi yönetimi

Oyun ayar dosyalarını düzenleme

Büyük oyun veri setleriyle çalışma araçları

Strateji oyunu geliştirme için erken arayüz sistemleri

Bu Repository'nin Amacı

Bu repository, editörün erken geliştirme aşamasındaki bir sürümünü korumak amacıyla oluşturulmuştur.

Şunlar için kullanılabilir:

Arşiv amaçlı saklama

Geliştirme geçmişini inceleme

Editörün erken mimarisini inceleme
